# TP2 - Workflow GitHub Actions

Ce projet contient le code source d'une application Android.

## Déclenchement du workflow

Pour tester le workflow CI/CD :

- Créer une branche depuis main : `feature/test-workflow`
- Ajouter une ligne ici dans le README pour déclencher le workflow.
- Commit et push sur GitHub.
- Créer une Pull Request vers main pour que GitHub Actions s’exécute.
